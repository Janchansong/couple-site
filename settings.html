如何获取同步服务器地址？（免费，约 5 分钟）
  1. 注册 Cloudflare（免费）
  2. 电脑打开命令行，进入本项目的 worker 文件夹
  3. 运行 npx wrangler login 登录一次
  4. 运行 npx wrangler kv namespace create SYNC_KV，把返回的 id 填进 worker/wrangler.toml
  5. 运行 npx wrangler deploy，得到地址如 https://couple-sync.xxx.workers.dev
  6. 把该地址填到上方「同步服务器地址」，同步码发给老婆

上线公网 🌐

不需要你公司电脑一直开着。下面都是云服务器托管（24 小时在线），部署一次即可，关机、下班老婆也能访问。

推荐：一键上云（GitHub Pages）

双击运行 deploy-cloud.bat，登录 GitHub 后自动推到云端。地址形如 https://你的用户名.github.io/couple-site/

方式 B：Netlify 云托管（约 2 分钟）

双击 deploy-netlify.bat，把 zip 拖到网页，获得 xxx.netlify.app 链接。同样不需要你电脑开着。

方式 C：Cloudflare Pages 云托管

登录 Cloudflare → Pages → 上传 zip。免费、全球 CDN。

⚠️ 本地 python server.py 或局域网地址只适合临时测试，公司电脑关机就打不开了。请用上面云托管方案。
